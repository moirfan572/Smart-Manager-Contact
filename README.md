# Smart-Manager-Contact
This is a Smart Manager Contact built using Java.it uses File Handling to Save Contacts Permanently.
*key Features
- Permanent Storage: Contacts Ko ContactData.txt File Mein Save Karta Hai.
- Full CRUD Operations: Aap Contacts ko Add,View,Search,Update,Delete kar Sakte hain.
- Smart Search: Naam Se Kisi Bhi Contact ko Turant Dhoondhein.
- Data Persistence: Program Shuru Hote Hi Purana Data Apne Aap Load Ho Jata hai.
* Tech Stack
* Language: Java(JDK 21).
* Concepts: OOps(Encapsulation).
* Collection Framework(ArrayList).
* File Handling(BufferedReader,PrintWriter).
* Exception Handling(try-Catch).
